# Stack
+ Also called 'container adapter'
+ By default implemented using deque containers
+ Stack can be implemented on any underlying container that provides following operations:
	+ `back()`
	+ `size()`
	+ `empty()`
	+ `push_back()`
	+ `pop_back()`

### Time complexities of `stack` functions:
#### O(1)
+ `push()`
+ `pop()`
+ `top()`
+ `size()`
+ `empty()`

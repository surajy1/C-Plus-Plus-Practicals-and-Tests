# Queue
+ Queue can be implemented on any underlying container that provides following functions:
	+ `empty()`
	+ `size()`
	+ `front()`
	+ `back()`
	+ `push_back()`
	+ `push_front()`

### Time complexities of `Queue` functions:
#### O(1)
+ `push()`
+ `pop()`
+ `front()`
+ `back()`
+ `empty()`
+ `size()`
